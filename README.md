# SushidaClone
html, css, jsの勉強を目的として作成したリポジトリ<br>
提示された文章通りにタイピングしていくもので正誤判定や時間制限がある．

お題の文章の内容のAPIは参考元(以下参照)と同じものを使っています．<br>
著者名やそれらの名言を返すものらしいです．
